# 青春林田行旅 - 博客系统

这是一个受万能青年旅店风格启发的现代博客系统，结合了东方禅意与独立音乐的美学风格。

## 设计理念

整个设计以"Inside The Cable Temple"为灵感，融合了以下元素：

- **自然基调**：采用低饱和的墨绿、苔藓绿色系，搭配浅米白或灰白作为背景，传递静谧与生机并存的氛围
- **独特排版**：中文采用手写书法体感，英文使用简洁无衬线字体，形成东西方元素的碰撞
- **复古质感**：整体风格偏向复古做旧，加入轻微噪点和纸质纹理，模仿老照片或褪色海报效果
- **简约留白**：大量留白，以极简主义风格为基础，通过细节纹理丰富层次

## 功能特点

- 文章发布、编辑与管理
- 评论系统
- 标签分类
- 用户管理
- 响应式设计，适配各种设备

## 技术栈

- **前端**：Vue 3、Vuex、Vue Router
- **后端**：Node.js、Express、MongoDB
- **API**：RESTful API

## 如何启动

1. 确保已安装Node.js和MongoDB
2. 在根目录运行 `start.bat` 脚本
3. 前端将在 http://localhost:8080 运行
4. 后端将在 http://localhost:3003 运行

## 视觉风格

系统的视觉风格遵循以下配色方案：

- **主色调**：浅米白 (#f5f5f0)、深墨绿 (#2c3e2e)
- **强调色**：锈红色 (#8b4c39)、靛蓝/灰蓝色 (#45556c)
- **中性色**：近黑色 (#232321)、暖白色 (#fffdf7)
